DynamicNews - Real-Time News Platform

DynamicNews is a real-time, modern news platform that provides short, summarized, and category-wise news updates. Users can filter articles by categories, sort the news by latest or oldest, and interact through likes and comments. The platform includes an admin dashboard for managing posts and monitoring user activity.

StayHub - Home/Villa Renting Website

StayHub is a full-stack renting platform where property owners can list their homes and villas. Built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript, it allows users to browse properties, view detailed descriptions, and explore locations using the integrated map feature. The platform includes secure authentication and a clean user-friendly interface.

Simon Game

A classic color-pattern memory game built using HTML, CSS, and JavaScript. The game generates a random sequence of flashing buttons, and the player's task is to repeat the pattern correctly. The game becomes progressively harder as the sequence length increases, helping players improve focus and memory skills.
